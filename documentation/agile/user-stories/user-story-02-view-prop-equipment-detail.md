**About**
As a **Production Company User** I can **view detailed information about a prop or equipment item** so that **I can assess its suitability for my production**

**Acceptance Criteria**

- [ ] Given I am on an item detail page, I can see a full description.
- [ ] Then I can see clear hire pricing information.
- [ ] Then I can see images that accurately represent the item.
- [ ] Then I can see an option to add the item to my basket.
